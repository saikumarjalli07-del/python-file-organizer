# Python File Organizer

## Project Overview

This project is a Python automation script that organizes files into different categories such as Images, Documents, Videos, and Others.

## Features

* User Input Support
* Automatic File Categorization
* Folder Creation
* File Movement
* Exception Handling
* Logging of Operations

## Technologies Used

* Python
* OS Module
* shutil Module
* datetime Module

## How It Works

1. User enters a folder path.
2. The program scans all files in the folder.
3. Files are categorized based on their extensions.
4. New folders are created automatically.
5. Files are moved to their respective folders.
6. All operations are logged in a log file.

## Sample Output

Moved: photo.jpg -> Images
Moved: report.pdf -> Documents
Moved: movie.mp4 -> Videos

File organization completed.
